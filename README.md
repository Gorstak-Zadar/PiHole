# 🕳️ PiHole (Ad Blocker)

> PowerShell **system-wide ad blocker** for Windows — uses DNS policy and persistent routes. Downloads EasyList, EasyPrivacy, AdGuard filters.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 📥 **Filter Lists** | EasyList, EasyPrivacy, AdGuard Base |
| 🔧 **DNS Policy** | `DnsPolicyConfig\BlockAdDomains` |
| 📋 **Persistent Routes** | Blocks ad IPs via routing |
| ⏰ **Update** | 24-hour update interval |
| 📝 **Logging** | `%TEMP%\AdBlocker.log` |

---

## 📋 Requirements

| Requirement | Details |
|-------------|---------|
| **OS** | Windows 10/11 |
| **PowerShell** | 5.1+ |
| **Privileges** | Administrator |
| **Network** | Internet for filter downloads |

---

## 🚀 Usage

```powershell
.\Pihole.ps1
```
